Marketplace de venta de productos orgánicos

HTML5 - CSS3 - Javascript - Tailwind- React - Vite - Spring Framework - Java - MySQL - Maven - Hibernate - Eureka - Junit 5 - Mockito - API RESTful

La presente aplicación web surge como respuesta a una necesidad real y cotidiana de las personas que buscan mantener una alimentación sana. El desafío radica en la dificultad para encontrar locales que oferten productos orgánicos y saludables. En la mayoría de los casos, su ubicación se conoce a través del boca a boca. Nuestra propuesta consiste en una aplicación que conecta a los usuarios con los vendedores de estos productos. De esta manera, se ofrece seguridad, transparencia y comodidad a quienes buscan mejorar su calidad de vida a través de la alimentación. La aplicación facilita el acceso a una amplia variedad de productos. En definitiva, la aplicación se posiciona como una herramienta útil y confiable para quienes buscan mejorar su salud y bienestar a través de una alimentación sana.

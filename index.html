Projects

News Website

Java - MySQL - HTML5 - CSS3 - Bootstrap - Thymeleaf - Spring - Spring Security - JPA - Maven

This project was created by me, it is a practice of an exercise proposed in the full stack programming course, the exercise asked me to make a news web page for the company EGG (it is the one that offers the full stack course) where there are 3 roles , a user, an administrator and a journalist, the administrator has access to the list of users, he can unsubscribe them, change journalist salary and change their role, the journalist can create news but not unsubscribe them and the normal user can only view News. the design was created to my liking, I have given it a touch of simplicity and I tried to make a comfortable interface for the user.

Health Website "Vitalidapp"

Java - MySQL - HTML5 - CSS3 - Javascript - Bootstrap - Thymeleaf - Spring Framework - Spring Security - JPA - Maven

This project was created with a group of 6 people (including me). Its purpose is to help self-employed doctors to register and care for patients, keep their medical records in an updated database and save diagnoses. The current problem in hospitals is that the medical history is lost very often and many times there is no order, so this page would be helping to solve part of the problem. My work here was mainly BackEnd but I've also been involved in FrontEnd not to mention Front to Back bindings and also Back to DataBase bindings. A little addition, I learned a little about three agile methodologies which are (Technical Lead, Scrum Master and Agile Delivery Manager).

JavaCrypter

HTML5 - CSS3 - Bootstrap - Javascript

This is a challenge provided by Alura's "ONE" program in collaboration with Oracle. The idea behind the challenge is to put into practice what has been learned during the program and create a more "normal" encryptor where the user should be able to enter a text and encrypt it, or if it is already encrypted, decrypt it. I aimed to give it an aesthetically pleasing design for the human eye with a unique color palette while also meeting certain requirements. For instance, users should not be able to input texts with special characters or accents. Additionally, the button that copies the returned text should also be considered.

News Website

Health Website "Vitalidapp"

Java - MySQL - HTML5 - CSS3 - Javascript - Bootstrap - Thymeleaf - Spring Framework - Spring Security - JPA - Maven

This web platform helps self-employed doctors to register and care for patients, keep their medical records in an updated database and save diagnoses. The platform solves the current problem in hospitals, where medical records are often lost and there is no clear order. The platform provides a safe and efficient solution for self-employed doctors to manage their patients.

JavaCrypter

HTML5 - CSS3 - Bootstrap - Javascript

This web platform is a challenge provided by the Alura "ONE" program in conjunction with Oracle. The idea of the challenge is to put into practice what was learned during the program and create an encryptor where the user must be able to enter a text and encrypt it or, if it is already encrypted, decrypt it. It has an aesthetically pleasing design with a unique color palette. It has certain limitations, the user cannot enter texts with special characters, accents, or uppercase letters.

Marketplace for the sale of organic products

HTML5 - CSS3 - Javascript - Tailwind- React - Vite - Spring Framework - Java - MySQL - Maven - Hibernate - Eureka - Junit 5 - Mockito - API RESTful

This web application arises as a response to a real and everyday need of people who seek to maintain a healthy diet. The challenge lies in the difficulty of finding stores that offer organic and healthy products. In most cases, their location is known through word of mouth. Our proposal consists of an application that connects users with the sellers of these products. In this way, it offers security, transparency and convenience to those who seek to improve their quality of life through food. The application facilitates access to a wide variety of products. In short, the application positions itself as a useful and reliable tool for those who seek to improve their health and well-being through a healthy diet.

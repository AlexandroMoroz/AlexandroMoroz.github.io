About Me

Hello! I'm Alex, also had nicknames like "Morita" or "Rusito", a web developer in training, I studied 7 years in a technical school in the electromechanical branch and then finished high school with a focus on fine arts. I started in programming months after finishing my high school studies, I studied in the Argentina Program 4.0 course and in a full-stack course with the company EGG COOPERATION, to this day I have been studying in this field for over 1 year. Throughout my life, I have worked in many sectors, such as metallurgy, carpentry, call centers, postal services, logistics, etc. Thanks to this experience, nowadays I think in a completely different way. I am a very calm person and I love working in teams, I am very dedicated to what I do and it is very difficult for me to leave a problem halfway, I generally do not rest until I find the solution.


Projects

News Website

Java - MySQL - HTML5 - CSS3 - Bootstrap - Thymeleaf - Spring - Spring Security - JPA - Maven

This website offers a platform for users to stay informed about the latest news, journalists to share their stories with the world, and administrators to control the website's content. Users can browse current news. Journalists can create accounts to publish news, edit their articles, and collaborate with other journalists. Administrators have access to the tools they need to manage the website, such as creating news and managing users.

Health Website "Vitalidapp"

Java - MySQL - HTML5 - CSS3 - Javascript - Bootstrap - Thymeleaf - Spring Framework - Spring Security - JPA - Maven

This web platform helps self-employed doctors to register and care for patients, keep their medical records in an updated database and save diagnoses. The platform solves the current problem in hospitals, where medical records are often lost and there is no clear order. The platform provides a safe and efficient solution for self-employed doctors to manage their patients.

JavaCrypter

HTML5 - CSS3 - Bootstrap - Javascript

This web platform is a challenge provided by the Alura "ONE" program in conjunction with Oracle. The idea of the challenge is to put into practice what was learned during the program and create an encryptor where the user must be able to enter a text and encrypt it or, if it is already encrypted, decrypt it. It has an aesthetically pleasing design with a unique color palette. It has certain limitations, the user cannot enter texts with special characters, accents, or uppercase letters.

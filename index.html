Justina.io

HTML5 - CSS3 - Javascript - React - Tailwind - Figma - Java - Spring Boot - MySQL - JWT - Inteligencia Artificial

Justina.io es una plataforma innovadora desarrollada para facilitar y agilizar la integración del complejo ecosistema de la salud.

Este proyecto fue parte de una hackatón organizada por No Country, en la cual mi equipo se destacó, logrando obtener el segundo lugar.

Justina.io tiene como objetivo centralizar y optimizar la gestión de información crítica de los pacientes, permitiendo una interacción eficiente entre diversos actores del sistema de salud, como profesionales médicos, pacientes, centros de salud, y entidades gubernamentales.

Entre sus funcionalidades más destacadas, Justina.io incluye un sistema de gestión de usuarios con roles específicos, asegurando que cada tipo de usuario tenga acceso a las herramientas y datos adecuados para sus necesidades. Además, la plataforma cuenta con una agenda integrada para la gestión de turnos médicos, facilitando la coordinación entre médicos y pacientes.

También se incorporó una avanzada funcionalidad para el registro de tratamientos y emisión de recetas, soportada por inteligencia artificial, lo que permite a los médicos capturar y gestionar información clínica de manera precisa y eficiente. Todo esto está envuelto en una interfaz omnicanal, asegurando una experiencia de usuario coherente y accesible desde cualquier dispositivo.

Justina.io

HTML5 - CSS3 - Javascript - React - Tailwind - Figma - Java - Spring Boot - MySQL - JWT - Artificial Intelligence

Justina.io is an innovative platform developed to facilitate and streamline the integration of the complex healthcare ecosystem.

This project was part of a hackathon organized by No Country, in which my team stood out, achieving second place.

Justina.io aims to centralize and optimize the management of critical patient information, enabling efficient interaction between various actors in the healthcare system, such as medical professionals, patients, healthcare centers, and governmental entities.

Among its most prominent features, Justina.io includes a user management system with specific roles, ensuring that each type of user has access to the appropriate tools and data for their needs. Additionally, the platform features an integrated schedule for managing medical appointments, facilitating coordination between doctors and patients.

An advanced functionality for recording treatments and issuing prescriptions, supported by artificial intelligence, has also been incorporated, allowing doctors to capture and manage clinical information accurately and efficiently. All of this is wrapped in an omnichannel interface, ensuring a consistent and accessible user experience across any device.

JavaCrypter

HTML5 - CSS3 - Bootstrap - Javascript

Esta plataforma web es un challenge proporcionado por el programa "ONE" de Alura en conjunto con Oracle. La idea del challenge es poner en práctica lo aprendido durante el programa y crear un encriptador en donde el usuario debe poder ingresar un texto y encriptarlo o en caso de ya estar encriptado, desencriptarlo. Tiene un diseño estéticamente agradable con una paleta de colores única. Tiene ciertos impedimentos, el usuario no puede ingresar textos con caracteres especiales, acentos o mayúsculas.
